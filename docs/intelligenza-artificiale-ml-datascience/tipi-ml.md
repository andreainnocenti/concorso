# Tipi di Machine Learning

## 📚 Supervised Learning
Apprendimento da dati etichettati.

## 🔍 Unsupervised Learning
Apprendimento senza etichette (clustering, PCA).

## 🎯 Semi-Supervised Learning
Pochi dati etichettati + molti non etichettati.

## 🎮 Reinforcement Learning
Apprendimento tramite reward/punishment.

## 🔄 Transfer Learning
Riutilizzare modelli pre-addestrati.
