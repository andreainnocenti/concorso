# Paradigmi e Linguaggi di Programmazione

## 📖 Introduzione

I paradigmi di programmazione sono approcci fondamentali alla risoluzione di problemi attraverso il codice. Ogni linguaggio supporta uno o più paradigmi.

## 🎯 Paradigmi Principali

### Programmazione Imperativa
Sequenza di comandi che modificano lo stato del programma.

### Programmazione Orientata agli Oggetti (OOP)
Organizzazione del codice in oggetti con dati e comportamenti.

### Programmazione Funzionale
Computazione come valutazione di funzioni matematiche.

### Programmazione Dichiarativa
Descrizione di "cosa" fare, non "come" farlo.

## 💻 Linguaggi Moderni

- **Python**: multi-paradigma, dinamico
- **Java**: OOP, tipizzato staticamente
- **JavaScript**: multi-paradigma, dinamico
- **Rust**: systems programming, memory-safe
- **Go**: concorrenza, semplicità
- **Haskell**: funzionale puro
